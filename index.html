TABLES

  • <.tr> is for one Table Row <./tr>
  • <.td> is for Table Data (like one cell) <./td>
  • <.th> is for one Table Head (like a heading) <./th>
  • <.tb> is for one Table Body (body content) <./tb>
  • <.tf> is for one Table Foot (footer) <./tf>
  • border = number : pixels define the size of the frame surrounding the table
    These are shown by adding border=1 (pixel size 1) next to <.table>
  • cellspacing = "number" : size of the space between two cells (shown through a percentage value or pixels)

  • <.label> Enter something: <./label>
  • Input box: <.input type = "text" name="" value=""> This will create a box to enter input after/next to the text
  • Submit button: <.input type="submit" name=""> This creates a submit box
  • Note: there are lots of different things you can put as the input type = "__"
  • Text area (a box to write in) syntax: <.textarea name="name" rows="#" cols="#"><./textarea>
    • rows = how tall, cols = how wide your text area should be
